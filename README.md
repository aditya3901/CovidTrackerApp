# CovidTrackerApp
Covid Tracker App 
<br>Overview : 

    1. Covid Stats of India 
    2. Covid Stats of India - State Wise
<br>Steps : 

    1. Calling Covid API using Android Volley
    2. JSON Parsing
    3. Implement RecyclerView
    4. Set Data to RecyclerView using custom Layout and Adapter
